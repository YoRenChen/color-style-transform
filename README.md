# colorStyleTransform

明暗风色彩转换搭建

## 说明

技术栈：vuecli/ vue/ less

```(filePath)
├── public // 文件放置处
├── src
│   ├── App.vue
│   ├── assets
│   ├── color.js
│   ├── components
│   ├── global.less
│   ├── main.js // 入口
│   ├── styles // 样式存放
│   └── utils
├── vue.config.js
```

可以直接执行`npm run color`进行生成样式文件
